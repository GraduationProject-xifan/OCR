组长：彭博

一。项目背景：
   当今社会，网络逐步融入大家的日常生活中，通过网络，人们组织起来了属于自己的社交网络，而在线聊天室，能够给你一个与朋友/陌生人一个面对面的机会。
   1.结识新朋友，来自全国各地的朋友，趣味相投？聊聊天吧
   2.专属聊天室，工作小组，学生班级讨论，情侣私语，我们将为您提供专属聊天室让你畅所欲言！
   3.在线登陆，无需下载体积庞大的客户端，打开浏览器即可登录
二。系统运行环境：
   全平台，有浏览器即可登录
三。项目功能模块：
   1.用户登录验证功能模块
   2.聊天室以及聊天功能模块
   3.聊天内容以及用户管理功能模块
   4.聊天页面动态刷新功能模块
   5.私聊以及好友功能模块
   6.后台系统管理模块
四。项目详细计划以及分工
